# 动态规划

```
有一排深渊法师，他们的护盾值用数组 a 表示；
你可以对深渊法师这样破盾：
1) 用重击破盾，每消耗1MP破1点护盾
2) 如果有两个相邻的深渊法师他们的元素不同，且护盾都不为0，你可以对他们俩使用高天之歌，消耗xMP破所有护盾
问最少消耗的MP？
下面输入的 els 表示深渊法师的元素，I 表示冰，W 表示水，F 表示火

输入：
a = [4, 8, 10, 2, 15, 2]
x = 9
els = WIFFII
```
